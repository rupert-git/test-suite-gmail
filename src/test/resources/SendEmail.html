Validation of basic email elements when composing an email

Example - warning shown

When composing an email message some basic address validation should occur. This warns the user if they are about to send an email with no recipients, or a badly formed recepient address.

Condition Outcome message
Invalid recipient provided (invalidAddress) Please make sure that all addresses are properly formed
No recipient provided Please specify at least one recipient

Example - successful send

Following on from the example above, if recipients are provided correctly then the email should send successfully, even if the email subject or email body are left blank.

Condition Outcome message
Correct recipient provided (validAddress@mailinator.com) Your message has been sent
Email subject left blank Your message has been sent
Email body left blank Your message has been sent
